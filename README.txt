		Тестове завдання для PHP Developer

			Завдання
	Розробити проект для управління користувачами. 
	Реалізувати базу даних користувачів в якій є 2 таблиці:
	          users:
			id
			name
			email
			country_id
	          countries:
			id
			country

	Реалізувати методи по роботі з базою з п.1, які дозволяють додавати, видаляти, оновлювати та  переглядати
 	список користувачів.

	Реалізувати просте UI для роботи з методами з п.2.
 
	Логувати видалення оновлення та створення, в файл, за допомогою бібліотеки Monolog

	Результатом повинне бути посилання на репозиторій з проектом

			Вимоги до коду
Проект повинен бути написаний без використання фреймворків
Для автозагрузки та підключення бібліотек повинен використовуватися composer
В корні проекта повинен лежати дамп БД
Код повинен бути написаний в ООП стилі.
Проект повинен коректно працювати на php 5.6+.
Перевагою буде дотримання стандартів PSR-1, PSR-2
